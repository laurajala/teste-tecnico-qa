# Critérios de Aceite

## Cadastro de usuário

Dado que o usuário esteja na tela de cadastro
Quando preencher os campos obrigatórios corretamente
Então o sistema deve criar a conta com sucesso

---

## Login válido

Dado que o usuário possua cadastro
Quando informar email e senha válidos
Então o sistema deve acessar a plataforma

---

## Adicionar produto ao carrinho

Dado que o usuário esteja autenticado
Quando adicionar um produto ao carrinho
Então o produto deve aparecer no carrinho
