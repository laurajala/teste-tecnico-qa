# História do Usuário

## Cadastro de usuário

Como visitante da plataforma,
quero realizar meu cadastro,
para acessar funcionalidades exclusivas do sistema.

---

## Login

Como usuário cadastrado,
quero acessar minha conta,
para utilizar as funcionalidades da plataforma.

---

## Carrinho de compras

Como usuário autenticado,
quero adicionar produtos ao carrinho,
para realizar futuras compras.
